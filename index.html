Social Post Studio

Social Post Studio is a private social media scheduling tool for preparing video posts before publishing them to connected creator accounts.

The app helps users organise videos, captions, hashtags, thumbnails, privacy settings, and platform-specific posting details in one place.

What The App Does

  • Prepares video posts for social media platforms.
  • Stores separate captions and tags for each platform.
  • Supports thumbnails and cover settings where platforms allow them.
  • Lets the user schedule posts for later instead of posting immediately.
  • Shows the status of scheduled, completed, skipped, or failed posting attempts.

How TikTok Is Used

TikTok Login Kit is used so the user can connect their own TikTok account. The app requests basic account information only to confirm which TikTok account is connected.

TikTok Content Posting API access is requested so the user can upload their own selected video from Social Post Studio to their own TikTok account.

Supported Workflow

  1. The user opens Social Post Studio on their own computer.
  2. The user connects their social account using the platform's official login flow.
  3. The user selects a video file they own or are authorised to post.
  4. The user writes captions, tags, and settings for the platform.
  5. The user schedules or starts the upload from inside Social Post Studio.

Who Uses It

Social Post Studio is currently used by The Bedlam Collective to manage and prepare promotional social media video posts for its own channels.

Data And Privacy

Social Post Studio stores account tokens and draft post data locally on the user's computer unless the user chooses to send a post to a connected platform.
